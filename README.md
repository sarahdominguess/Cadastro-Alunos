# Cadastro de Alunos
Esse projeto é um código em python realizado por Sarah Domingues para a matéria de Principios de Construção de Algoritmos. Ele visa a realização do cadastro de alunos em um arquivo CSV e utilizando pandas.

## Dependências:

* Como esse código utiliza pandas, é necessário instalar esse pacote. Para isso, no terminal do seu computador, digite pip install pandas

  ## Exemplos de Funcionalidades:

  * INSERIR aluno
  * REMOVER aluno
  * PESQUISAR aluno
  * EDITAR dados
